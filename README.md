# Итоговая Аттестация
Ловягин Евгений Евгеньевич
**Описание:** Это приложение позволяет пользователям генерировать случайные пароли с возможностью настройки длины и типов символов, а также сохранять историю сгенерированных паролей.

## Примеры использования
1. Запустите приложение.
2. Настройте параметры пароля.
3. Нажмите "Сгенерировать пароль" для получения нового пароля.

Пароли будут сохраняться в файле history.json.
